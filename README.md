# Alexa Voice Service Server

> Node.js web server for interacting with the [Alexa Voice Service](https://developer.amazon.com/appsandservices/solutions/alexa/alexa-voice-service).

# Usage

Install dependencies

```
npm install
```

Run

```
npm start
```
